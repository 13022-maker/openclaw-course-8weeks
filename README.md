# OpenClaw 校園 AI×軟工工作坊（8 週）— 作業素材包

這是一套給高中/高職「正在學程式設計」學生使用的作業素材包（繁體中文、校園情境）。

## 你會拿到什麼
- 每週一個資料夾 `week01` ~ `week08`
- **每週都包含**：
  - `README.md`：作業目標、步驟、驗收標準
  - `data/`：範例輸入檔案
  - `expected/`：範例輸出（對照用）
  - `src/`：起始骨架程式（Python 為主；第 7 週含 C++）
  - `tests/`：簡單測試/驗證腳本
  - `rubric.md`：評分重點

## 建議環境
- Python 3.10+（本包以 Python 為主）
- （可選）C++ 編譯器：clang++ 或 g++（第 7 週用）

## 使用方式（老師/助教）
- 每週把當週資料夾發給學生即可
- 或整包發出，讓學生按週完成

> 註：若要整合 OpenClaw（agent/排程/skills）實作，可在課程環境另外提供 OpenClaw 安裝與帳號/模型設定。
